## Usage

<DocSection>
  The `Badge` component composes a `View` wrapper and a `Text` child via the `TextClassContext` for seamless variant-matched text styles.

### Variants

<DocSubsection>
  Choose from `default`, `secondary`, `destructive`, and `outline`.
  <CompPreview
    code={`<Badge variant="default"><Text>New</Text></Badge>
<Badge variant="secondary"><Text>Beta</Text></Badge>
<Badge variant="destructive"><Text>Error</Text></Badge>
<Badge variant="outline"><Text>Draft</Text></Badge>`}
  >
    <BadgeVariantsInlinePreview />
  </CompPreview>
</DocSubsection>

### With Status Text

  <DocSubsection>
    Use badges inline to indicate statuses on list items or cards.
    <CompPreview code={`<Badge variant="secondary">
  <Text>In Progress</Text>
</Badge>
<Badge variant="default">
  <Text>Shipped</Text>
</Badge>`}>
      <BadgeStatusInlinePreview />
    </CompPreview>
  </DocSubsection>
</DocSection>
