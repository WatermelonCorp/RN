## Usage

<DocSection>
  The `Avatar` primitive uses `@rn-primitives/avatar` for lazy loading with an accessible `AvatarFallback` shown before the image resolves.

### With Image

<DocSubsection>
  Provide a `src` to load a remote image for the avatar.
  <CompPreview
    code={`<Avatar>
  <AvatarImage src="https://github.com/shadcn.png" />
  <AvatarFallback>
    <Text>SC</Text>
  </AvatarFallback>
</Avatar>`}
  >
    <AvatarImageInlinePreview />
  </CompPreview>
</DocSubsection>

### Fallback Only

<DocSubsection>
  When no image is provided, `AvatarFallback` always renders.
  <CompPreview
    code={`<Avatar>
  <AvatarFallback>
    <Text>JD</Text>
  </AvatarFallback>
</Avatar>`}
  >
    <AvatarFallbackInlinePreview />
  </CompPreview>
</DocSubsection>

### Custom Sizes

  <DocSubsection>
    Override the default 40×40 size with NativeWind classes.
    <CompPreview code={`<Avatar className="h-16 w-16">
  <AvatarFallback>
    <Text variant="large">AB</Text>
  </AvatarFallback>
</Avatar>`}>
      <AvatarSizeInlinePreview />
    </CompPreview>
  </DocSubsection>
</DocSection>
