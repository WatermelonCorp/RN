## Usage

<DocSection>
  Use `SpotlightButton` when you want a single call-to-action to feel more cinematic than a standard button.

### Hero CTA

  <DocSubsection>
    Pair the component with a short badge and a concise action label.

    <CompPreview
      code={`<SpotlightButton size="lg" badge="Beta">\n  Launch animation\n</SpotlightButton>`}
    >
      <SpotlightButtonDemo />
    </CompPreview>

  </DocSubsection>

### Neutral Surface

  <DocSubsection>
    Switch to the `neutral` variant when the button sits inside a denser card or dashboard surface.

    <CompPreview
      code={`<SpotlightButton variant="neutral" badge="Update">\n  View release notes\n</SpotlightButton>`}
    >
      <div className="flex min-h-[220px] items-center justify-center rounded-b-xl bg-[linear-gradient(180deg,hsl(var(--background)),color-mix(in_oklab,hsl(var(--muted))_55%,transparent))] p-6">
        <SpotlightButton variant="neutral" badge="Update">
          View release notes
        </SpotlightButton>
      </div>
    </CompPreview>

  </DocSubsection>

### Custom Copy

  <DocSubsection>
    Use the `badge` prop to tune urgency without rewriting the rest of the component structure.

    <CompPreview
      code={`<SpotlightButton badge="Launch">\n  Open preview build\n</SpotlightButton>`}
    >
      <SpotlightButtonInlineDemo />
    </CompPreview>

  </DocSubsection>
</DocSection>
