## Usage

<DocSection>
  The `Input` component wraps React Native's `TextInput` with NativeWind v4 styles and a variant API for visual modes.

### Basic Usage

<DocSubsection>
  A standard text input with placeholder and focus ring.
  <CompPreview code={`<Input placeholder="Enter your name" />`}>
    <InputBasicInlinePreview />
  </CompPreview>
</DocSubsection>

### Ghost Variant

<DocSubsection>
  Use the `ghost` variant for borderless inputs on coloured backgrounds.
  <CompPreview code={`<Input variant="ghost" placeholder="Search..." />`}>
    <InputGhostInlinePreview />
  </CompPreview>
</DocSubsection>

### Disabled

<DocSubsection>
  Pass `editable={false}` to disable the input.
  <CompPreview
    code={`<Input placeholder="Read-only value" editable={false} />`}
  >
    <InputDisabledInlinePreview />
  </CompPreview>
</DocSubsection>

### Secure Entry

  <DocSubsection>
    Use `secureTextEntry` for password inputs.
    <CompPreview code={`<Input placeholder="Password" secureTextEntry />`}>
      <InputSecureInlinePreview />
    </CompPreview>
  </DocSubsection>
</DocSection>
