## Usage

<DocSection>
  <div>
    The <code>Button</code> component wraps a React Native <code>Pressable</code>{" "}
    and shares text styling through the registry <code>Text</code> primitive.
  </div>

  ### Basic Usage

  <DocSubsection>
    <div>Use the default button for primary actions.</div>
    <CodeBlock language="tsx" title="Basic Usage">
      {`<Button>
  <Text>Continue</Text>
</Button>`}
    </CodeBlock>
  </DocSubsection>

  ### Variants

  <DocSubsection>
    <div>
      Choose from <code>default</code>, <code>secondary</code>,{" "}
      <code>outline</code>, <code>ghost</code>, <code>destructive</code>, and{" "}
      <code>link</code>.
    </div>
    <CodeBlock language="tsx" title="Variants">
      {`<Button variant="default"><Text>Primary</Text></Button>
<Button variant="secondary"><Text>Secondary</Text></Button>
<Button variant="outline"><Text>Outline</Text></Button>
<Button variant="ghost"><Text>Ghost</Text></Button>
<Button variant="destructive"><Text>Delete</Text></Button>`}
    </CodeBlock>
  </DocSubsection>

  ### Sizes

  <DocSubsection>
    <div>
      Match the control to the surface with <code>sm</code>,{" "}
      <code>default</code>, <code>lg</code>, or <code>icon</code>.
    </div>
    <CodeBlock language="tsx" title="Sizes">
      {`<Button size="sm"><Text>Small</Text></Button>
<Button size="default"><Text>Default</Text></Button>
<Button size="lg"><Text>Large</Text></Button>`}
    </CodeBlock>
  </DocSubsection>

  ### With Text

  <DocSubsection>
    <div>
      Because the registry shares class names with the nested <code>Text</code>{" "}
      element, icon and label compositions stay visually consistent.
    </div>
    <CodeBlock language="tsx" title="Composed Button">
      {`<Button variant="outline" size="lg">
  <Text>Open Preview</Text>
</Button>`}
    </CodeBlock>
  </DocSubsection>
</DocSection>
