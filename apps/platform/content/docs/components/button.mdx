## Usage

<DocSection>
  The `Button` component wraps a React Native `Pressable` and shares text styling through the registry `Text` primitive.

### Basic Usage

<DocSubsection>
  Use the default button for primary actions.
  <CompPreview
    code={`<Button>
  <Text>Continue</Text>
</Button>`}
  >
    <ButtonBasicInlinePreview />
  </CompPreview>
</DocSubsection>

### Variants

<DocSubsection>
  Choose from `default`, `secondary`, `outline`, `ghost`, `destructive`, and
  `link`.
  <CompPreview
    code={`<Button variant="default"><Text>Primary</Text></Button>
<Button variant="secondary"><Text>Secondary</Text></Button>
<Button variant="outline"><Text>Outline</Text></Button>
<Button variant="ghost"><Text>Ghost</Text></Button>
<Button variant="destructive"><Text>Delete</Text></Button>`}
  >
    <ButtonVariantsInlinePreview />
  </CompPreview>
</DocSubsection>

### Sizes

<DocSubsection>
  Match the control to the surface with `sm`, `default`, `lg`, or `icon`.
  <CompPreview
    code={`<Button size="sm"><Text>Small</Text></Button>
<Button size="default"><Text>Default</Text></Button>
<Button size="lg"><Text>Large</Text></Button>`}
  >
    <ButtonSizesInlinePreview />
  </CompPreview>
</DocSubsection>

### With Text

  <DocSubsection>
    Because the registry shares class names with the nested `Text` element, icon and label compositions stay visually consistent.
    <CompPreview code={`<Button variant="outline" size="lg">
  <Text>Open Preview</Text>
</Button>`}>
      <ButtonTextInlinePreview />
    </CompPreview>
  </DocSubsection>
</DocSection>

## API Reference

<DocSection>
  `Button` extends all supported props from React Native `Pressable` and adds
  design-system variants.
  <ApiTable
    rows={[
      {
        prop: "variant",
        type: `"default" | "destructive" | "outline" | "secondary" | "ghost" | "link"`,

      },
      {
        prop: "size",
        type: `"default" | "sm" | "lg" | "icon"`,
        default: `"default"`,
        description: "Controls button height, padding, and icon sizing.",
      },
      {
        prop: "disabled",
        type: "boolean",
        default: "false",

      },
      {
        prop: "className",

        description: "Extends the NativeWind classes applied to the control.",
      },
    ]}
  />
